# Nuclipp Website 🚀

The official premium web platform for **Nuclipp**, a comprehensive creative agency specializing in YouTube Automation, Graphic Design, and Web Development.

## Overview
This platform was built as a single-page application experience with a stunning glassmorphic, "Claude-inspired" aesthetic. It serves to showcase Nuclipp's elite service offerings, impressive portfolio of YouTube channels and Instagram pages, and the brilliant minds behind the agency.

## Core Features
*   **Premium Glassmorphic Design:** Advanced CSS techniques providing frosted glass effects (`backdrop-filter`), dynamic glowing orbs, and fluid micro-animations.
*   **Responsive Layout:** Fully optimized for desktop, tablet, and mobile displays with dynamic staggered timelines and flexible grid systems.
*   **YouTube Automation Focus:** Specialized service area highlighting end-to-end video automation pipelines.
*   **Integrated Contact Engine:** A seamless lead-capture form securely bridging client inquiries to `business.nuclipp@gmail.com`.

## Tech Stack
*   **Framework:** Next.js (App Router)
*   **Styling:** Pure CSS Modules + Custom Design Variables
*   **Icons:** Lucide-React
*   **Deployment:** Ready for Vercel

## Local Development
To run this project locally:

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
